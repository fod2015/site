    + 最近在做什麽？

    2017新年過后，我對這個網站進行了部分內容更新，包括作品及簡歷部分等等。

    我的二手交換網站上綫了，由我設計及開發部份Data使用AngularJS管理及維護，雖然目前以Airsoft爲主，本著物盡其用的原則，這些暫時閒置的物品，或者對其他人有所幫助。

    最近喜歡上 Country Music，我架設了一個基於 SHOUTcast 的網絡電台，每天又多了一個聲音向全世界播放電台熱播鄉村歌曲。同時我的電台主頁也上線了歡迎訪問：http://wanghao.hk/radio/

    個人網站改版后，我搭建了獨立的Wordpress博客，可以不受限制的分享所見所聞所感。歡迎訪問：http://blog.wanghao.hk/

    + 現在介紹下自己

    我十分熱愛網頁開發與設計，喜歡創作與分享，從構思到實踐，手寫代碼建立豐富的網頁，將視覺設計與內容量身結合，向用戶展示產品的價值。我從未刻意把它視為一份工作，而是生活的一部分，簡單又快樂；

    我認爲興趣是工作的重要源動力之一，可以激發出更多的火花和靈感，如果可以將更多的興趣結合起來，我相信可以做到更出色；

    在工作中一直努力平衡令人滿意的工作效率與工作質量，具備成熟的心理素質，承受高強度的工作壓力；

    + 工作技能:我擅長什麽？

    具備豐富的響應式網站（Responsive Web Development）設計與開發經驗；

    熟練運用 HTML/CSS/SCSS/SASS 頁面排版、佈局重構，解决不同及新舊浏览器版本的頁面兼容性问题；

    熟練運用 HTML5, CSS3, SVG, Javascript, jQuery 等加入創意動畫，豐富頁面效果；

    具備豐富的 eDM newsletters 推廣電郵設計與開發經驗；

    具備針對移動設備端網站優化技巧，以及APP前端頁面開發經驗；

    熟練使用 Photoshop, Dreamweaver, Sublime 等設計與開發軟件，

    熟悉CMS內容管理系統，Bootstrap/AngularJS/Json/Ajax，熟悉 WordPress 搭建、部署與使用，基本SEO技巧；

    具備網站基本視覺設計和網站Banner設計經驗。

    + 除此之外，你還會些什麽？

    我有一定的寫作及圖文編譯能力，喜歡閱讀熱愛分享，擅長追蹤可靠的信息來源；

    我對互聯網和移動互聯網的新興產品非常感興趣，如360Cam, VR, 關注最新的網頁前端技術，亦有多類軟件的使用經驗；

    在過往的工作中，我管理并熟悉論壇社區、博客、微博等互動產品，具備一定的社交網絡使用與推廣經驗；

    我喜歡運動攝影（如GoPro），也會使用一些非线性编辑软件 Premiere, Sony Vegas 等进行视频、音頻的剪辑與加工。

    偶爾將想法付諸實踐拍攝有意思的短片（在2015年《星球大戰：原力覺醒》電影上映時片方舉辦的光劍接力視頻比賽活動中，我的參與短片獲得二等獎）。

    + 我的學位與學歷

    2005年9月于北京入讀中國政法大學，至2009年7月畢業，經受過4年專業嚴格的法學本科教育及訓練，我目前擁有法學學士學位。

    我已參加過IELTS與CET-6英文水平測試，其中IELTS分數6.0（2013年3月），CET-6成績507（標準綫420），此外，我識聽廣東話并可做簡單交流，具有良好的國語听说读写能力。

    + 我是否可以立即就職，薪酬要求？

    我尊重雇主及雇傭單位的薪酬福利制度，我目前在職，亦正尋找更具挑戰的工作機會，如果閣下對我感興趣，歡迎透過郵件作進一步交談，我抱以十分的熱忱期待您的邀請。

    + 我未來的職業規劃

    興趣驅動著我從事自己喜歡的工作，並使我將所喜歡的工作做得更好，我看重一份穩定的工作能夠挑戰、鍛煉自己，我熱愛網頁設計、開發網站、寫代碼，如果未來有機會能結合其他興趣一起工作，將是一件非常美好的事情。

    + 我对雇主有哪些期待？

    強有力的領導核心，充滿創意與凝聚力的團隊；一台配置不錯的電腦，一把舒適的座椅，如果還有更多請立即告訴我。
